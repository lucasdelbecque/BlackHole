# BlackHole

Simulateur de trou noir en ray tracing GPU (Metal + Schwarzschild).

## Build & lancement

```bash
cmake -S . -B build && cmake --build build && ./build/BlackHole
```

## Contrôles

| Action | Geste |
|---|---|
| Orbite | Clic gauche + drag |
| Zoom | Pinch trackpad ou scroll deux doigts |
| Grille | Touche `G` |
